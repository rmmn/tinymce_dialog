TinyMCE Quick Start Guide

Подключение

                
                    <script src="path/tinymce-settings.js"></script>
                    <script>
                        $('textarea_id').tinymce({
                            ...
                            menu: {
                                tc: {
                                    title: 'TinyComments',
                                    items: 'addcomment showcomments deleteallconversations '
                                },
                                blocks: {
                                    title: "Блоки",
                                    items: 'blocks'
                                },
                                visualization: {
                                    title: "Визуализация",
                                    items: 'visual'
                                }
                            },
                            menubar: 'file edit view insert format tools table tc help blocks visualization',
                            setup: (editor) => {
                                editor.ui.registry.addMenuItem('blocks', {
                                    text: 'Вставить список товаров',
                                    onAction: () => OnDialogButtonPressed(editor)
                                });
                                editor.ui.registry.addMenuItem('visual', {
                                    text: 'Показать пример списка',
                                    onAction: () => OnVisualizationPressed(editor)
                                });
                            }
                            ...
                        });
                    </script>

Настройка и изменение надписей в окне

Редактируется в файле tiny-settings.js

                
                        params = {
                            DialogTitle: string,
                            VizualizationTitle: string,
                            // Кнопки в диалоге
                            Buttons: {
                                // Текст на главной кнопке
                                PrimaryLabel: string,

                                // Текст на кнопке отмены
                                CancelLabel: string
                            },

                            // Вкладки
                            Tabs: {
                                GeneralTitle: string,
                                ProductsTitle: string,
                                AdditionalsTitle: string
                            },

                            // Здесь указываются начальные значения полей (опционально, можно оставить пустым) 
                            initData: {
                                input_name: "value"
                            }
                        }
